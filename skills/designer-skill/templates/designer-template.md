## 人物提示词格式

文件：assets/designer.md

第一部分：5张核心主图设计规范（2:3比例，美学+点击双核心）
主图核心目标：在搜索流中实现“3秒心动，点击进入”，以产品美学抓眼球，以核心价值留用户，5张图形成一套完整的「心动→了解→沉浸→信任→下单」迷你美学叙事线。

1.  **主图1：心动首图（搜索核心主图）**
    - 核心定位：以产品美学抓眼，3秒让用户产生“这个产品很好看、很高级”的第一印象，同时精准承接搜索需求
    - 内容规范：产品全景主体（占画面60%-70%，居中C位，无任何遮挡）+ 品牌LOGO（角落极简呈现，不抢产品风头）+ 1句极简核心价值slogan（不超过10个字，贴合产品气质）+ 1个精致化核心利益点（无爆炸贴、无廉价色块）
    - 美学核心要求：
      背景采用与产品色调适配的低干扰质感背景（哑光肌理、渐变柔色、极简场景），拒绝杂乱背景；用定制化光影塑造产品核心质感，突出产品的设计线条、材质肌理；slogan与利益点放在画面留白区，字体与产品气质完全匹配，字号克制，绝不遮挡产品主体；平台缩略图缩小后，依然能清晰看清产品形态与整体质感，实现“缩略图里也好看”。
    - 合规提示：禁用极限词，利益点真实可兑现，与详情页完全一致。

2.  **主图2：质感细节图**
    - 核心定位：放大产品的细节美学，让用户感知产品的品质与用心，强化核心价值
    - 内容规范：产品核心设计/材质/结构的微距特写 + 1个核心卖点的极简注解（不超过8个字，仅点题，不赘述）
    - 美学核心要求：
      用特写镜头+精准光影，放大产品最动人的细节（木纹肌理、釉面光泽、面料编织感、金属倒角、面料缝线），让用户产生“可触摸”的质感联想；画面极简，仅保留特写主体与留白，文字仅作为画面点缀，不破坏细节的呈现；用画面本身传递卖点，而非文字解释，例：保温杯的锁温卖点，用杯口硅胶圈的微距特写呈现，而非大段文字说明。
    - 合规提示：所有细节均为产品真实存在，无虚假美化。

3.  **主图3：场景氛围感图**
    - 核心定位：让用户产生场景代入与情绪共鸣，感知产品在生活中的美感与价值
    - 内容规范：产品核心使用场景的氛围感画面 + 产品在场景中的自然呈现 + 无多余文字，仅用画面传递情绪
    - 美学核心要求：
      场景完全贴合目标人群的真实生活状态，光影、色调、道具均与产品调性统一，例：香薰产品用夜晚卧室的柔暖光影场景，咖啡器具用清晨阳台的自然光场景；产品在场景中自然融入，而非生硬摆放，画面有故事感、有生活气息，让用户产生“我也想拥有这样的生活”的向往；拒绝夸张摆拍、违和网红图，整体氛围松弛、真实、有感染力。

4.  **主图4：信任质感图**
    - 核心定位：用高级感的方式建立用户信任，打消品质顾虑，拒绝生硬的资质堆砌
    - 内容规范：核心权威资质/专利/检测数据的极简美学化呈现 + 产品局部呼应画面
    - 美学核心要求：
      拒绝直接堆砌证书原图，将核心信任状转化为极简的视觉元素：专利用线条化图标呈现，检测数据用柔和的进度环/极简数字呈现，资质用精致的极简边框排版；画面背景与整体色调统一，搭配产品的局部细节作为视觉呼应，整体风格克制、专业、有质感，让用户在舒适的视觉体验中接收信任信息，而非被生硬的证书劝退。
    - 合规提示：所有资质、数据真实有效，标注专利号/检测机构，禁止伪造信息。

5.  **主图5：权益收尾图**
    - 核心定位：温柔引导下单，清晰呈现到手权益，兼顾美感与转化，拒绝廉价促销感
    - 内容规范：产品全规格/全套装的整齐陈列 + 极简优惠政策 + 售后保障承诺 + 赠品清单（如有）
    - 美学核心要求：
      产品与赠品采用对称/错落的美学化陈列，光影统一，质感一致；优惠信息用品牌点缀色做极简突出，无大红大绿的爆炸贴、无满屏的促销文字；售后保障用极简线性图标呈现，整体画面干净、清晰、有秩序感，让用户一眼看清到手权益，同时不破坏全案的高级质感。

## 六、第二部分：详情页10屏分屏设计规范（2:3比例，美学叙事+转化闭环）
10屏详情页严格遵循「产品美学叙事线」，情绪层层递进，屏与屏之间视觉衔接自然，每一屏兼具独立的美学表达与明确的转化目标，彻底告别生硬的卖点堆砌与数据轰炸，让用户在审美沉浸中完成购买决策。

### 第1屏：首屏KV氛围感屏
- 核心定位：承接主图点击，用电影级氛围感画面留住用户，完成“初见心动”的第一印象，奠定全案的美学基调
- 设计目标：让用户点进来的瞬间，就被画面的美感与情绪抓住，瞬间明确产品的核心气质与价值，无跳出意愿
- 内容规范：产品核心视觉主体 + 品牌LOGO + 1句有情绪、有价值的核心slogan + 3个极简价值标签（仅关键词，无长句）
- 美学核心要求：
  采用电影级构图与光影，打造有氛围感的主视觉，用画面传递产品的核心气质，例：护肤品用柔雾漫反射光影打造温润高级感，户外装备用旷野自然光打造自由松弛感；产品占据画面视觉C位，无任何遮挡，背景与产品色调、气质完全统一；slogan字体与画面风格融为一体，放在画面留白区，价值标签用极简线性图标呈现，无生硬色块；整体画面有呼吸感、有故事感、有强烈的视觉记忆点，拒绝满屏文字堆砌。
- 避坑提示：禁止为了放卖点遮挡产品主体，slogan拒绝生硬的广告话术，优先传递情绪与核心价值。

### 第2屏：用户情绪共鸣屏
- 核心定位：用温柔的画面与共情的表达，戳中目标用户的核心痛点，建立情绪共鸣，而非制造焦虑
- 设计目标：让用户产生“这说的就是我”的强烈共鸣，自然引出产品的解决方案，无抵触心理
- 内容规范：3-4个核心痛点的情绪化呈现，每个痛点对应一句共情式短句，拒绝夸张吐槽、负面画面
- 美学核心要求：
  采用与痛点情绪匹配的氛围感画面，例：“换季皮肤泛红”用女生轻抚脸颊的温柔特写，而非烂脸的负面图片；“厨房收纳混乱”用杂乱厨房的柔和光影画面，而非夸张的垃圾堆场景；每个痛点单独分区，画面有疏有密，文案用柔和的字体放在留白区，语气共情而非指责；整体色调与全案统一，用画面传递情绪，而非文字轰炸，让用户在情绪共鸣中自然接受后续的产品解决方案。
- 避坑提示：痛点必须精准匹配目标人群，拒绝泛化痛点，禁止负面、夸张、引发不适的画面。

### 第3屏：核心价值解决方案屏
- 核心定位：针对用户痛点，用产品的核心美学与功能，给出温柔的解决方案，强化产品不可替代的核心价值
- 设计目标：让用户瞬间明白“这个产品能完美解决我的困扰，而且它很好看、很适合我”
- 内容规范：1-3个核心主打卖点，每个卖点对应一个用户痛点，用“产品细节+效果呈现”的方式表达，拒绝生硬的功能罗列
- 美学核心要求：
  以产品核心功能的特写画面为主视觉，用光影突出产品的核心设计与质感；每个卖点对应一个产品特写镜头，用画面呈现功能效果，而非大段文字解释；例：“静音破壁机”用产品运行时，水面无波动的特写画面呈现静音效果，而非“低至40分贝”的生硬数字；文案仅作为画面的注解，极简短句，放在留白区，字体与画面风格统一；整体画面有层次、有重点，让用户在欣赏产品细节的同时，自然get到核心价值。
- 避坑提示：禁止只讲功能不讲效果，禁止脱离产品的文字堆砌，所有卖点均为产品真实具备的属性。

### 第4屏：产品细节美学屏
- 核心定位：放大产品的细节之美，让用户感知产品的品质与用心，全方位提升产品价值感
- 设计目标：让用户感受到“这个产品连细节都做得这么好，品质一定不会差”，强化物超所值的认知
- 内容规范：3-5个产品核心细节，按用户关注度排序，每个细节对应一个产品的设计巧思/材质优势/使用体验
- 美学核心要求：
  采用分区块微距特写排版，每个细节对应一个独立的画面分区，用精准的光影放大产品的材质肌理、设计线条、工艺细节；例：实木家具的木纹肌理、陶瓷餐具的釉面光泽、服装的缝线工艺、家电的倒角设计；每个细节仅搭配1句极简注解，无长段文字；画面分区清晰，节奏明快，有疏有密，整体色调与光影风格完全统一，像一本产品细节美学画册，让用户沉浸式感受产品的品质。
- 避坑提示：禁止与核心卖点重复，仅展示有用户价值、有美感的细节，拒绝无关紧要的无效细节堆砌。

### 第5屏：核心参数美学化屏
- 核心定位：用美学化的方式呈现产品核心参数，告别冰冷的表格与数字堆砌，让用户轻松接收硬核信息
- 设计目标：让用户在舒适的视觉体验中，清晰了解产品的硬核实力，打消“虚假宣传”的顾虑，同时不破坏全案的美学基调
- 内容规范：仅保留3-5个用户最关心的核心参数/核心成分/核心配置，非核心参数统一放在最后一屏附录
- 美学核心要求：
  拒绝生硬的Excel表格、数字罗列，将参数与产品本身的结构、设计结合起来；例：家电的容量、功率，用产品的剖面线条图，搭配极简标注呈现；护肤品的核心成分，用成分分子的极简线条图标，搭配产品瓶身特写呈现；核心数据用与整体风格统一的字体，放在对应的产品细节旁，用柔和的线条、图标做视觉引导；整体画面干净、专业、有秩序感，与全案的色调、字体完全统一，让参数自然融入产品美学，而非独立于画面之外。
- 合规提示：所有参数、数据均来自产品真实信息，标注来源，无篡改、无夸大。

### 第6屏：全场景沉浸体验屏
- 核心定位：用多场景的氛围感画面，让用户全面代入产品的使用场景，拓展产品价值边界，强化购买欲望
- 设计目标：让用户感受到“这个产品能融入我生活的方方面面，买了它能提升我的生活质感”
- 内容规范：1个核心高频场景 + 3个延伸适配场景，每个场景对应产品的自然使用状态，无多余文字
- 美学核心要求：
  采用电影感分镜式排版，每个场景对应一个独立的画面区块，光影、色调与全案统一，场景之间有自然的情绪衔接；每个场景都贴合目标人群的真实生活，产品在场景中自然融入，有生活气息、有故事感，而非生硬摆拍；例：餐桌椅的场景，涵盖日常三餐、朋友聚餐、居家办公、亲子手工；画面整体松弛、治愈、有代入感，让用户在沉浸式的场景浏览中，不断强化“我需要这个产品”的认知。
- 避坑提示：场景必须真实贴合目标人群，禁止虚构不切实际的使用场景，拒绝违和、夸张的摆拍画面。

### 第7屏：差异化价值美学屏
- 核心定位：用克制、客观的方式，呈现产品的独家差异化优势，强化不可替代性，拒绝恶意拉踩、生硬对比
- 设计目标：让用户一眼看清“这款产品和普通款的核心区别，为什么它更适合我”，同时不破坏全案的高级质感
- 内容规范：3-4个本品与同品类产品的核心差异化优势，仅客观对比用户真正关心的维度，无无效对比
- 美学核心要求：
  拒绝左右分栏的生硬对比、红绿色块的拉踩式排版，采用“本品细节特写+优势价值呈现”的方式，聚焦本品的独家优势；例：“食品级不锈钢材质”用本品材质的微距特写，搭配极简的“耐腐防锈、无重金属析出”注解，而非“普通款易生锈、有安全隐患”的恶意拉踩；画面以本品的细节美学为核心，差异化优势用极简的线性图标+短句呈现，整体风格克制、客观、高级，让用户自主感知产品的优势，而非被强行灌输。
- 合规提示：对比内容真实客观，有参数支撑，禁止虚假对比、恶意诋毁竞品。

### 第8屏：品牌与信任美学屏
- 核心定位：用有温度、有质感的方式，传递品牌理念与信任背书，全面建立用户信任，拒绝生硬的品牌故事与证书堆砌
- 设计目标：让用户相信“这是一个有态度、有品质、靠谱的品牌，买它很放心”
- 内容规范：品牌核心理念 + 核心权威资质/专利/奖项 + 市场口碑极简呈现
- 美学核心要求：
  品牌理念用有温度的画面呈现，例：原创设计品牌，用设计师手绘稿、产品打样过程的氛围感画面；有机食品品牌，用产地种植环境的自然光影画面；资质、专利、奖项拒绝原图堆砌，用极简的相框式排版、线性图标呈现，放在画面留白区，与品牌画面自然融合；整体画面有温度、有故事感、有信任感，色调与全案统一，让用户在了解品牌的过程中，进一步强化对产品的认可，而非被大段的品牌故事劝退。
- 合规提示：所有资质、专利、奖项真实有效，标注相关编号与发证机构，禁止伪造信息。

### 第9屏：使用与售后安心屏
- 核心定位：用清晰、温柔的方式，呈现使用须知与售后保障，彻底打消用户的决策顾虑，降低下单门槛
- 设计目标：让用户明确“买了之后有保障，使用无门槛，完全可以放心下单”
- 内容规范：极简使用步骤/适配条件 + 核心售后保障服务 + 禁忌事项/不适用人群
- 美学核心要求：
  使用步骤用极简的线性插画/产品实拍分镜呈现，拒绝大段文字说明；售后保障用温柔的品牌色、极简图标呈现，核心保障项放大突出，放在画面上半区；禁忌事项用柔和的提示框呈现，重点内容加粗，语气友好，而非生硬的警告；整体画面干净、清晰、有秩序感，字体、色调与全案完全统一，让用户在舒适的阅读体验中，全面打消售后顾虑，而非被密密麻麻的条款劝退。
- 避坑提示：禁止隐瞒禁忌事项与不适用人群，售后政策与店铺公示完全一致，无虚假承诺。

### 第10屏：下单引导收尾屏
- 核心定位：用温柔的方式完成临门一脚的转化引导，清晰呈现下单权益，同时呼应全案的美学基调，完成整个叙事闭环
- 设计目标：让用户一眼看清“怎么选、花多少钱、拿到什么”，打消最后的选择顾虑，引导立即下单
- 内容规范：全SKU规格美学化陈列 + 核心优惠政策 + 限时福利/赠品清单 + 品牌slogan收尾
- 美学核心要求：
  SKU产品采用整齐、有美感的陈列方式，光影统一，质感一致，不同规格的差异用极简短句标注，清晰易懂；优惠信息用品牌点缀色做极简突出，无廉价促销感、无爆炸贴；赠品用实拍特写呈现，与产品风格统一；画面底部用一句呼应首屏的slogan收尾，完成整个美学叙事的闭环；整体画面干净、有秩序、有温度，让用户在舒适的视觉体验中，完成最终的下单决策。
- 避坑提示：禁止虚假优惠，SKU差异清晰明确，避免用户下单后产生纠纷。

## 七、补充美学落地适配规则
