# 产品信息收集产出模板

输出文件：outputs/01-production-information-collection-template.md

---

### 模块1：产品基础身份信息（详情页基础框架核心素材）
精准提取产品唯一身份标识，对应详情页标题、SKU区、参数表头基础信息，要求无任何模糊表述：
1.  产品全称（含系列、版本全称，禁止简称）
2.  品牌名称（含品牌所属主体、品牌slogan，如有）
3.  电商精准类目（精确到主流电商平台三级类目，例：家电>厨房家电>空气炸锅）
4.  核心规格/型号、SKU核心差异项（例：不同容量、配色、配置的核心区别）
5.  产地、生产/委托方主体信息
6.  保质期/有效期、质保周期、存储要求
7.  市场定价区间、核心价格带定位（例：入门性价比款、中端品质款、高端旗舰款）

### 模块2：产品核心硬性数据（详情页参数区、信任状核心素材）
仅提取**客观可溯源、有明确依据**的量化数据，禁止主观加工，所有数据必须标注单位与来源，对应详情页参数表、检测报告展示、数据可视化设计：
1.  核心性能参数（例：家电的功率、容量、能效等级；食品的营养成分、核心配料占比；数码产品的芯片型号、续航时长；护肤品的核心成分含量）
2.  权威检测数据（第三方机构检测报告数据、国标/行标达标情况、有效率/转化率等实验数据，必须标注检测机构与时间）
3.  核心配置/成分全清单（按重要性排序，标注核心成分/配置的核心作用）
4.  行业对标数据（与国标、同品类行业均值、同价位主流竞品的核心参数差异）

### 模块3：产品核心卖点与差异化优势（详情页首屏、核心转化模块核心素材）
按转化优先级排序，严格区分客观可验证卖点与商家宣传卖点，每个卖点必须对应解决的用户痛点，对应详情页核心卖点板块、对比板块设计：
1.  核心主打卖点（1-3个，产品最核心、不可替代、直击用户核心痛点的独家优势，为详情页首屏核心叙事主题）
2.  次级辅助卖点（3-5个，补充核心卖点、提升产品价值感的配套优势，按用户关注度排序）
3.  竞品独家差异化优势（与同价位、同品类头部竞品相比，独有的、无法被复刻的核心竞争力，为详情页竞品对比板块核心内容）
4.  用户可感知价值承诺（产品能给用户带来的具象化、可感知的使用结果，禁止夸大与违规表述）

### 模块4：目标受众与用户画像（详情页用户共情、场景化设计核心素材）
精准锚定用户群体，对应详情页痛点唤醒、场景化叙事、文案语气风格设计：
1.  核心目标人群（1-2类，产品核心付费用户，精准到年龄区间、性别、消费层级、职业特征、生活状态）
2.  次级覆盖人群（2-3类，产品可触达的延伸用户群体，例：送礼人群、家庭共用人群）
3.  核心用户痛点与消费顾虑（目标人群未被满足的核心需求、购买该类产品的核心顾虑、使用中的高频痛点，每个痛点必须对应产品的一个卖点）
4.  用户购买决策关键因素（目标用户下单时最看重的核心维度，按优先级排序，例：安全性、功效、性价比、颜值、品牌、售后）

### 模块5：使用场景与操作说明（详情页场景化设计、使用教程板块核心素材）
覆盖产品全场景适配，对应详情页场景化视觉设计、使用步骤演示板块：
1.  核心使用场景（1-3个，产品最主要、最高频的使用场景，含场景环境、使用人群、使用目的）
2.  延伸使用场景（2-4个，产品可适配的多元化场景，拓展用户需求边界，提升产品价值感）
3.  核心使用流程/操作方法（极简步骤化呈现，适配详情页图文/短视频教程设计）
4.  适配条件与使用门槛（产品使用所需的环境、设备、技能要求，例：家电电压要求、护肤品肤质适配范围）
5.  禁忌事项与不适用人群/场景（明确标注产品禁止使用的场景、不适配的人群，规避售后风险与合规问题）

### 模块6：品牌信任背书与售后保障（详情页信任状、决策门槛降低板块核心素材）
提取可提升用户信任度的全部信息，对应详情页品牌介绍、资质展示、售后保障板块：
1.  品牌核心优势（品牌历史、行业地位、核心专利技术、品牌理念、供应链优势）
2.  权威资质认证（3C认证、有机认证、专利证书、行业奖项、国际国内合规认证，标注认证机构与有效期）
3.  市场口碑与数据背书（全网累计销量、好评率、复购率、权威媒体/达人推荐、头部渠道合作背书）
4.  售后保障服务（退换货政策、运费险、质保服务、上门安装、专属客服、一对一指导等，降低用户决策门槛）

### 模块7：详情页设计专属补充信息（直接对接设计落地的专属内容，提前规避风险）
1.  **产品视觉核心亮点（必须基于参考产品图片）**：结合product-information中的产品参考图片，准确描述产品外观设计、配色方案、材质质感、结构亮点、LOGO位置和样式，适合详情页重点视觉呈现的核心元素，为设计师提供视觉聚焦方向，这是后续AI准确还原产品的关键
2.  合规红线风险提示（对素材中出现的违反《广告法》《电子商务法》及品类监管要求的内容，逐一标注，明确说明违规风险，禁止用于详情页宣传）
3.  详情页核心叙事逻辑建议（基于产品卖点、用户痛点，给出详情页内容排序优先级建议，例：痛点先行→卖点解决→数据佐证→场景展示→信任背书→售后保障）
4.  同品类避坑提示（同品类详情页高频违规点、用户差评集中的核心问题，提前规避设计与文案风险）
